## How to Use This

1. **Before studying:** Read the `mental-model.md` in each unit folder — it maps all topics and shows which questions each topic covers.
2. **For understanding:** Work through the notes files (01 → 05) in order. Each file starts with a real-world analogy before any technical definition.
3. **For exam prep:** Use the `README.md` in each notes folder — it has the full checklist, key diagrams to draw, and a quick-revision cheat sheet.
4. **For practice:** Read each question in the question bank, attempt an answer, then compare with the answers file.
